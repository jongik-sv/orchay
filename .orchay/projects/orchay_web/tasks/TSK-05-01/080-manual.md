## 1. 기능 개요

Task Detail Panel은 WBS 트리에서 Task를 선택했을 때 오른쪽 패널에 표시되는 상세 정보 영역입니다. Task의 기본 정보 조회, 인라인 편집, 진행 상태 확인 기능을 제공합니다.

### 1.1 주요 기능

| 기능 | 설명 |
|------|------|
| 기본 정보 표시 | Task ID, 제목, 카테고리, 우선순위, 담당자 표시 |
| 인라인 편집 | 제목, 우선순위, 담당자 실시간 편집 |
| 진행 상태 | 현재 워크플로우 상태 및 진행률 시각화 |
| 낙관적 업데이트 | 편집 시 UI 즉시 반영 후 서버 동기화 |

---

## 2. 사용 방법

### 2.1 Task 선택

1. WBS 트리 패널에서 Task 노드를 클릭합니다.
2. 오른쪽 Detail Panel에 해당 Task의 상세 정보가 표시됩니다.

### 2.2 제목 편집

1. Task 제목 영역을 클릭합니다.
2. 입력 필드가 활성화됩니다.
3. 새 제목을 입력합니다 (1-200자).
4. Enter 키를 누르거나 필드 외부를 클릭하여 저장합니다.
5. Escape 키로 편집을 취소할 수 있습니다.

### 2.3 우선순위 변경

1. 우선순위 Dropdown을 클릭합니다.
2. 4개 옵션 중 선택합니다:
   - **긴급 (Critical)**: 빨간색
   - **높음 (High)**: 주황색
   - **보통 (Medium)**: 파란색
   - **낮음 (Low)**: 회색
3. 선택 즉시 저장됩니다.

### 2.4 담당자 변경

1. 담당자 Dropdown을 클릭합니다.
2. 프로젝트 팀원 목록에서 선택합니다.
3. "미지정"을 선택하여 담당자를 해제할 수 있습니다.

## 3. 화면 구성

### 3.1 컴포넌트 구조

```
TaskDetailPanel (컨테이너)
├── TaskBasicInfo (기본 정보)
│   ├── Task ID Badge
│   ├── 제목 (인라인 편집)
│   ├── 카테고리 Tag
│   ├── 우선순위 Dropdown
│   └── 담당자 Dropdown
└── TaskProgress (진행 상태)
    ├── 현재 상태 Badge
    └── 워크플로우 단계 인디케이터
```

### 3.2 상태별 화면

| 상태 | 화면 |
|------|------|
| Task 미선택 | "왼쪽에서 Task를 선택하세요" 메시지 |
| 로딩 중 | Skeleton 로딩 표시 |
| 로드 실패 | 에러 메시지 + 재시도 버튼 |
| 정상 | Task 상세 정보 표시 |
